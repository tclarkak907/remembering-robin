# Robin Gail Clark Memorial Website

This is a static memorial website built from the photos and themed artwork in the surrounding `Moms Slideshow` folder.

## Local preview

Serve the `robin-memorial-site` folder as the web root. The celebration slideshow is embedded from the family's shared Google Drive copy.

## Publishing

The website is suitable for free static hosting. Keep the Google Drive slideshow sharing setting at "Anyone with the link" so visitors can watch without requesting access.
